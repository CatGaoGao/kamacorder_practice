# 面试题02.07.链表相交、160.链表相交

力扣链接：https://leetcode.cn/problems/intersection-of-two-linked-lists-lcci/

---

按自己思路完整写出来了，比较顺

---

**思路：**



1. a或b为空，返回空   
2. 统计a和b的长度   
3. 取a和b长度差的绝对值，用建立新指针从头节点开始遍历，把差值先走了，然后再一起走   
4. 若有交点则返回交点，否则最后返回空

---

## 代码

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        if(headA == NULL || headB == NULL){
            return NULL;
        }

        ListNode* cur1 = headA;
        int count1 = 0;
        ListNode* cur2 = headB;
        int count2 = 0;
        while(cur1){
            cur1 = cur1->next;
            count1++;
        }
        while(cur2){
            cur2 = cur2->next;
            count2++;
        }

        int cha = abs(count1-count2);
        ListNode* t1 = headA;
        ListNode* t2 = headB;

        if(count1 > count2){
            while(cha--){
                t1 = t1->next;
            }
            while(t1 != NULL && t1 != t2){
                t1 = t1->next;
                t2 = t2->next;
            }
            return t1;
        }
        else{
            while(cha--){
                t2 = t2->next;
            }
            while(t2 != NULL && t1 != t2){
                t1 = t1->next;
                t2 = t2->next;
            }
            return t1;
        }
        return NULL;
    }
};
```
