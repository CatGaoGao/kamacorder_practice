# 746.使用最小花费爬楼梯

力扣链接：https://leetcode.cn/problems/min-cost-climbing-stairs/

---

直接看了视频讲解，思路很清晰，爬楼梯消费版，希望下次看到类似的题型可以自己ac

---

**思路：**

1. **dp[i]含义：** 到达第 i 阶梯所用的最小花费，**i** 含义：第 i 级阶梯   
2. **递推公式** dp[i] = min(dp[i-1]+cost[i-1], dp[i-2]+cost[i-2])   
3. **初始化** dp[0] = 0, dp[1] = 0; 因为是到达第i阶梯的最小花费，还没跳不用花费
4. **遍历顺序** 从前往后遍历

---

## 代码

```cpp
class Solution {
public:
    int minCostClimbingStairs(vector<int>& cost) {
        // 定义dp数组
        int n = cost.size();
        vector<int> dp(n+1);

        // 初始化
        dp[0] = 0;
        dp[1] = 0;

        // 遍历
        for(int i = 2; i <= n; i++){
            dp[i] = min( dp[i-1] +cost[i-1] , dp[i-2] +cost[i-2]);
        }

        return dp[n];
    }
};
```
