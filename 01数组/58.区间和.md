# 58.区间和

卡码链接：https://kamacoder.com/problempage.php?pid=1070

---

**思路：**

1. 遍历，算出每个元素位置的累加和，存入数组   
2. 累加和数组根据下标互减
3. 注意起始点下标为0，以及不为0的分开处理

---

## 代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
  // 输入元素个数
  int n;
  cin >> n;

  // 输入元素值，同时计算累加和
  vector<int> nums(n);
  vector<int> sum(n);
  int he = 0;
  for (int i = 0; i < n; i++) {
    cin >> nums[i];
    he += nums[i];
    sum[i] = he;
  }

  int a, b; // 输入下标
  int ans = 0;

  while (cin >> a >> b) {
    // 计算和
    if(a == 0){ // 注意此种情况
      ans = sum[b];
    }
    else{
      ans = sum[b] - sum[a-1];
    }

    cout << ans << endl;
  }
}
```
