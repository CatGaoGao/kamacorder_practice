# 19.删除链表的倒数第N个节点

力扣链接：https://leetcode.cn/problems/remove-nth-node-from-end-of-list/

---

最开始按自己的思路写，一直有报错，在ai的帮助下以及看了答案自己改成功了，然后答案版本的也写了，其实思路一致的

---

**思路：**

虚拟头节点、快慢双指针

1. 头为空，返回空   
2. 快指针while循环n--，结束循环再走一步，因为慢指针只到目标数的前一位   
3. 快慢指针一起走，直到快为空   
4. 删除操作

---

## 代码

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        if(head == nullptr){
            return NULL;
        }

        ListNode* xnhead = new ListNode(0);
        xnhead->next = head;
        ListNode* fast = xnhead;
        ListNode* slow = xnhead;

        // 找到目标数
        while(n--){
            fast = fast->next;
        }
        fast = fast->next; // slow只到目标数前一个数

        while(fast){
            fast = fast->next;
            slow = slow->next; // 到达操作位置
        }

        ListNode* temp = slow->next;
        slow->next = slow->next->next;
        delete temp;

        return xnhead->next;
    }
};
```
