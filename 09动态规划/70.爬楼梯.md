# 70.爬楼梯

力扣链接：https://leetcode.cn/problems/climbing-stairs/

---

直接看的讲解视频，很容易理解，本质其实就是斐波那契数列

---

**思路：**

爬楼梯就是斐波那契数列

1. **dp[i]含义：** 到达第 i 阶梯的方法数量，**i** 含义：第 i 级阶梯   
2. **递推公式** dp[i] = dp[i-1] + dp[i-2] 走到第 i 阶梯，无非就是从 上一阶梯 或者 上两个阶梯 跨上来，所以到 i 的时候就是前两个阶梯的方法数相加   
3. **初始化** dp[1] = 1, dp[2] = 2; 没有定义dp[0]因为没有啥意义，从1开始
4. **遍历顺序** 从前往后遍历

---

## 代码

写的是空间复杂度为 O（1）的一版，只用到了三个变量，迭代

如果按照递推公式里的写，空间复杂度就是 O（n）

class Solution {
public: // 1/1 2/2 3/3 4/5 5/8...相当于斐波那契数列
    int climbStairs(int n) {
        if(n == 1) return 1;
        if(n == 2) return 2;
        
        int dp[3];
        dp[1] = 1;
        dp[2] = 2;

        for(int i = 3; i <= n; i++){
            int sum = dp[1] +dp[2];
            dp[1] = dp[2];
            dp[2] = sum;
        }

        return dp[2];
    }
};
