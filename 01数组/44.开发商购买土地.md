# 44.开发商购买土地

**思路：**

分行和列处理（横向、竖向分割）

1. 遍历，算出土地总和   
2. 分行和列分别计算，每算出一行/列，就用剩下的部分-当前已经累加起来的行/列 取绝对值，用min更新结果   

---

## 代码

```cpp
#include<iostream>
#include<vector>
#include<climits>
#include<cmath>
using namespace std;

int main(){
  int n,m;
  cin >> n >> m;
  vector<vector<int>> nums(n, vector<int>(m, 0));
  int sum = 0;
  int result = INT_MAX;

  for(int i = 0; i < n; i++){
    for(int j = 0; j < m; j++){
      cin >> nums[i][j];
      sum += nums[i][j];
    }
  }

  // 横向切割
  int count = 0;
  for(int i = 0; i < n; i++){
    for(int j = 0; j < m; j++){
      count += nums[i][j];
    }
    result = min(result, abs((sum-count) - count));
    // sum-count是剩余部分的行和，count是当前行和
    // 二者的差就是要求的最小差距
  }

  // 竖向切割
  int countt = 0;
  for(int j = 0; j < m; j++){
    for(int i = 0; i < n; i++){
      countt += nums[i][j];
    }
    result = min(result, abs((sum-countt) - countt));
  }

  cout << result;
  return 0;
}

```
