# 203.移除链表元素

力扣链接：https://leetcode.cn/problems/remove-linked-list-elements/

---

有思路的，但是循环条件写不对，要注意

---

**思路：**

虚拟头结点

1. 设定虚拟头结点指向头结点   
2. 用cur->next遍历，然后删目标节点   

---

## 代码

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* removeElements(ListNode* head, int val) {
        ListNode* temp = nullptr; // 创建临时节点
        ListNode* xnhead = new ListNode(0); // 用虚拟头节点做
        xnhead->next = head;
        ListNode* cur = xnhead;

        while (cur->next != nullptr) { // 注意此处循环条件
            if (cur->next->val == val) {
                // 因为要考虑到头节点?=val所以用cur->next来判断更方便
                temp = cur->next;
                cur->next = cur->next->next;
                delete temp;
            } else {
                cur = cur->next;
            }
        }

        // 把虚拟头删了
        head = xnhead->next;
        delete xnhead;
        return head;
    }
};
```
