# 242.有效的字母异位词

力扣链接：https://leetcode.cn/problems/valid-anagram/

---

思路想了一半，本来还设想用哈希表，结果想到后面发现也不对，看答案发现直接用数组做应该更方便简单

---

**思路：**

1. 创建数组存一组的字母出现次数   
2. 遍历另一组然后减去次数   
3. 遍历数组，出现不为0的数时return false

---

## 代码

```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        int record[26] = {0};
        for(int i = 0;i < s.size(); i++){
            record[s[i]-'a']++;  // 字母和a的差一定是在0-25之间
        }

        for(int j = 0;j < t.size(); j++){
            record[t[j]-'a']--;
        }

        for(int a = 0; a < 26; a++){
            if(record[a] != 0){
                return false;
            }
        }

        return true;
    }
};
```
