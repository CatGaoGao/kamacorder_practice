# 27.移除元素

力扣链接：https://leetcode.cn/problems/remove-element/

---

基础不好，暴力解法知道思路但是写完运行总是有错误，想不到一些细节
注意双指针是下标

---

**思路：**

设定快慢指针对应下标，移动`fast`为主,只要`fast`值不为指定数，就赋值给`slow`,然后`slow++`   

---

## 暴力解法  
两个for循环，时间复杂度O(n^2)

```cpp
class Solution {
public:   // 暴力解法
    int removeElement(vector<int>& nums, int val) {
        int k = nums.size();
        for (int i = 0; i < k; i++) {
            if (nums[i] == val) {
                for (int j = i+1; j < k; j++) {
                    nums[j-1] = nums[j];
                }// 前一位等于后一位，覆盖掉

                // 因为覆盖，后一位前移了，所以下标i减小了1
                i--;
                // 数组长度也减小了
                k--;
            }
        }
        return k;
    }
};
```

## 双指针，快慢指针   
时间复杂度O(n)

```cpp
class Solution {
public:   // 快慢指针
    int removeElement(vector<int>& nums, int val) {
        int slow = 0;
        for(int fast = 0; fast < nums.size(); fast++){
            if(nums[fast] != val){
                nums[slow] = nums[fast];
                // slow取等于val的说不清，直接取需要的
                slow++;
            }
        }
        return slow;
    }
};
```
